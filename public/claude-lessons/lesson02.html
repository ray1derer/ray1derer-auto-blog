핵심 개념

웹 인터페이스

브라우저만 있으면 어디서든 접속 가능한 기본 인터페이스입니다.

Claude Pro

더 많은 대화, 우선 접속, 새로운 기능을 먼저 체험할 수 있는 유료 플랜입니다.

대화 히스토리

이전 대화를 저장하고 검색하여 지속적인 작업이 가능합니다.

크로스 플랫폼

웹, iOS, Android에서 동일한 계정으로 동기화됩니다.

실습: 일주일 식단 계획 만들기

Step 1: 플랫폼별 접속

웹 브라우저: claude.ai 접속

모바일: 브라우저에서 claude.ai 북마크 추가

팁: 홈 화면에 바로가기 추가하면 앱처럼 사용 가능

Step 2: 개인 정보 입력

안녕하세요! 저를 위한 일주일 식단을 짜주세요.
- 나이: [당신의 나이]
- 목표: [다이어트/근육증가/건강유지]
- 알레르기: [있다면 작성]
- 선호 음식: [좋아하는 음식]
- 요리 실력: [초급/중급/고급]
- 예산: [주간 식비 예산]

Step 3: 상세 요청

다음 형식으로 만들어주세요:
1. 요일별 아침/점심/저녁 메뉴
2. 각 식사의 예상 칼로리
3. 장보기 리스트
4. 간단한 조리법
5. 식사 준비 팁

Step 4: 개인화 수정

생성된 식단을 검토하고 수정 요청:

좋네요! 그런데 몇 가지 수정하고 싶어요:
- 화요일 저녁을 좀 더 간단한 메뉴로
- 전체적으로 탄수화물을 조금 줄여주세요
- 주말엔 외식 1회 포함해주세요

Step 5: 저장과 활용

완성된 식단을 다양한 방법으로 저장:

  • 대화 북마크하기
  • 텍스트 복사해서 메모장에 저장
  • 스크린샷 찍기
  • 공유 링크 생성 (Pro 기능)

무료 vs Pro 비교

기능 무료 플랜 Claude Pro
일일 메시지 수 제한적 (약 10-30개) 5배 더 많은 메시지
대기 시간 피크 시간 대기 가능 우선 접속
새로운 기능 정식 출시 후 조기 액세스
대화 길이 표준 더 긴 대화 가능
가격 무료 월 $20

💡 Pro Tips

효율적인 Claude 사용을 위한 팁:

  • 대화 관리: 주제별로 새 대화 시작하기
  • 북마크 활용: 자주 참조하는 대화 저장
  • 모바일 최적화: 긴 텍스트는 음성 입력 활용
  • 오프라인 대비: 중요한 내용은 별도 저장
  • 플랜 선택: 사용 패턴 분석 후 Pro 결정
