학습 목표

  • 프로젝트 관리 방법론을 옵시디언에 통합하기
  • 작업 추적과 진행 상황 모니터링 시스템 구축
  • 팀 협업을 위한 워크플로우 설계
  • 프로젝트 대시보드와 리포트 자동화

프로젝트 관리 방법론

Agile/Scrum

스프린트 기반 반복 개발

  • 스프린트 계획
  • 일일 스탠드업
  • 회고 및 개선

Kanban

시각적 작업 흐름 관리

  • WIP 제한
  • 흐름 최적화
  • 지속적 전달

Waterfall

순차적 단계별 진행

  • 요구사항 분석
  • 설계 → 구현
  • 테스트 → 배포

Hybrid

복합 접근 방식

  • 유연한 적용
  • 맞춤형 프로세스
  • 최적화된 워크플로우

Weekly Report Generator 
```dataviewjs
// 주간 상태 리포트 자동 생성
const startOfWeek = moment().startOf('week');
const endOfWeek = moment().endOf('week');

// 완료된 작업 수집
const completedTasks = dv.pages('"Tasks"')
    .where(p => p.completed_date >= startOfWeek && 
                p.completed_date <= endOfWeek)
    .sort(p => p.completed_date, 'desc');

// 진행 중인 작업
const inProgressTasks = dv.pages('"Tasks"')
    .where(p => p.status == "In Progress");

// 이슈 및 블로커
const blockers = dv.pages('"Issues"')
    .where(p => p.priority == "Critical" && p.status != "Closed");

// 리포트 생성
dv.header(2, `주간 리포트 - ${startOfWeek.format("YYYY-MM-DD")} ~ ${endOfWeek.format("YYYY-MM-DD")}`);

dv.header(3, "📊 주요 지표");
dv.paragraph(`- 완료된 작업: ${completedTasks.length}개`);
dv.paragraph(`- 진행 중: ${inProgressTasks.length}개`);
dv.paragraph(`- 긴급 이슈: ${blockers.length}개`);

dv.header(3, "✅ 완료된 작업");
dv.table(["작업", "담당자", "완료일"], 
    completedTasks.map(t => [
        t.file.link,
        t.assignee,
        t.completed_date
    ])
);

dv.header(3, "🚨 주요 이슈");
if (blockers.length > 0) {
    dv.table(["이슈", "담당자", "마감일"],
        blockers.map(b => [
            b.file.link,
            b.assignee,
            b.due_date
        ])
    );
} else {
    dv.paragraph("*현재 긴급 이슈 없음*");
}
```

// Jira 이슈 가져오기 스크립트
const jiraUrl = "https://company.atlassian.net";
const project = "PROJ";

// JQL 쿼리
const jql = `project = ${project} AND sprint in openSprints()`;

// 이슈 동기화
async function syncJiraIssues() {
    const response = await fetch(`${jiraUrl}/rest/api/2/search?jql=${jql}`);
    const data = await response.json();
    
    for (const issue of data.issues) {
        const note = {
            title: `${issue.key}: ${issue.fields.summary}`,
            type: issue.fields.issuetype.name,
            status: issue.fields.status.name,
            assignee: issue.fields.assignee?.displayName,
            priority: issue.fields.priority.name,
            description: issue.fields.description
        };
        
        // 노트 생성 또는 업데이트
        await createOrUpdateNote(note);
    }
}
