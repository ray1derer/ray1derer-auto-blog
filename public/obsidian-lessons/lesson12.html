1. 일일 노트의 철학

왜 일일 노트인가?

두뇌 확장

매일의 생각과 아이디어를 외부 저장소에 보관

연결성

시간의 흐름 속에서 아이디어의 발전 추적

성장 기록

개인의 발전과 학습 과정을 문서화

인사이트 발견

패턴과 트렌드를 통한 자기 이해

"일일 노트는 단순한 일기가 아니라, 시간을 통해 진화하는 개인 지식 관리 시스템의 중추입니다."

📊 자동 통계

Dataview로 동적 대시보드 생성

```dataviewjs
// 이번 주 완료한 작업
const thisWeek = dv.pages('"Daily Notes"')
  .where(p => p.file.day >= dv.date("this week"))
  .file.tasks.where(t => t.completed);

// 생산성 차트
const productivity = dv.pages('"Daily Notes"')
  .where(p => p.file.day >= dv.date("this month"))
  .map(p => ({
    date: p.file.name,
    tasks: p.file.tasks.length,
    completed: p.file.tasks.where(t => t.completed).length
  }));

dv.table(["Date", "Total Tasks", "Completed", "Rate"],
  productivity.map(p => [
    p.date,
    p.tasks,
    p.completed,
    `${Math.round(p.completed/p.tasks*100)}%`
  ])
);
```
