섹션별 작성 가이드

Introduction

구조
  1. 일반적 배경 (General → Specific)
  2. 문제 제기
  3. 기존 연구의 한계
  4. 연구 목적과 기여
체크리스트
  • Hook이 있는가?
  • 연구의 중요성이 명확한가?
  • Research gap이 명시되었는가?
  • 연구 질문이 구체적인가?

Methods

포함 사항
  • 연구 설계 및 접근법
  • 참여자/표본 정보
  • 측정 도구 및 검증
  • 자료 수집 절차
  • 분석 방법
작성 팁
  • 재현 가능하도록 상세히
  • 과거 시제 사용
  • 윤리적 고려사항 포함

Results

구성 원칙
  • 연구 질문 순서대로
  • 기술통계 → 추론통계
  • 텍스트, 표, 그림 상호보완
  • 객관적 서술 (해석 X)

Discussion

논의 포인트
  1. 주요 발견 요약
  2. 이론적 기여
  3. 실무적 시사점
  4. 한계점과 미래 연구

통계 분석 연동

R Integration 
```r
# R 코드 블록 (R 플러그인 필요)
library(ggplot2)
data <- read.csv("research_data.csv")

# 기술통계
summary(data)

# 시각화
ggplot(data, aes(x=group, y=score, fill=condition)) +
  geom_boxplot() +
  theme_minimal() +
  labs(title="Group Comparison", 
       x="Group", 
       y="Score")

# 통계 검정
t.test(score ~ condition, data=data)
```

LaTeX 변환

# Pandoc을 통한 LaTeX 변환
pandoc paper.md -o paper.tex \
  --template=ieee-template.tex \
  --bibliography=references.bib \
  --csl=ieee.csl \
  --pdf-engine=xelatex

Word 문서 변환

# Word 형식으로 변환
pandoc paper.md -o paper.docx \
  --reference-doc=journal-template.docx \
  --bibliography=references.bib \
  --csl=apa-7th.csl

# 논문 투고 체크리스트 - {{Journal Name}}

## 📋 일반 요구사항
- [ ] 단어 수 제한 확인 (현재: {{word_count}}/{{limit}})
- [ ] 저자 정보 블라인드 처리
- [ ] 이해상충 선언문 작성
- [ ] 저자 기여도 명시
- [ ] 윤리 승인 번호 포함

## 📄 원고 준비
- [ ] 제목 페이지 별도 작성
- [ ] 초록 단어 수 확인 ({{abstract_words}}/250)
- [ ] 키워드 5-7개 선정
- [ ] 본문 형식 지침 준수
  - [ ] 폰트: Times New Roman 12pt
  - [ ] 줄 간격: Double
  - [ ] 여백: 1 inch
  - [ ] 페이지 번호

## 🖼️ 그림과 표
- [ ] 모든 그림 고해상도 (300 DPI 이상)
- [ ] 그림 파일 별도 제출
- [ ] 표 형식 통일
- [ ] 그림/표 번호와 설명

## 📚 참고문헌
- [ ] 저널 스타일 가이드 준수
- [ ] DOI 링크 포함
- [ ] 모든 인용 본문에 대응
- [ ] 알파벳 순서 정렬

## 💾 제출 파일
- [ ] 메인 원고 (저자정보 제외)
- [ ] 제목 페이지
- [ ] 그림 파일들
- [ ] 보충 자료
- [ ] 커버 레터

문제 해결

Zotero 연동이 안 돼요

  • Better BibTeX 설치 확인
  • 자동 내보내기 경로 확인
  • .bib 파일 권한 확인
  • Citations 플러그인 재시작

인용 형식이 잘못되었어요

  • CSL 파일 확인
  • BibTeX 항목 형식 검증
  • 플러그인 설정에서 스타일 변경

내보내기가 깨져요

  • Pandoc 설치 확인
  • 템플릿 파일 경로 확인
  • 인코딩 설정 (UTF-8)
  • 이미지 경로 절대/상대 확인
