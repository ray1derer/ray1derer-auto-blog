JavaScript 
```javascript {1,3-5}
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

console.log(fibonacci(10)); // 55
```

다양한 언어 지원

Python

def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)

SQL

SELECT 
    u.name,
    COUNT(o.id) as order_count,
    SUM(o.total) as total_spent
FROM users u
LEFT JOIN orders o ON u.id = o.user_id
GROUP BY u.id
HAVING COUNT(o.id) > 5
ORDER BY total_spent DESC;

Callouts와 Admonitions

기본 Callout 문법

Markdown 
> [!note]
> 이것은 기본 노트 callout입니다.

> [!tip] 유용한 팁
> 제목을 추가할 수도 있습니다.

> [!warning]- 접을 수 있는 경고
> 마이너스(-)를 추가하면 기본적으로 접혀있습니다.

> [!danger]+ 항상 열려있는 위험
> 플러스(+)를 추가하면 항상 열려있습니다.

Callout 종류

[!note] 일반 정보
[!abstract] 요약, 개요
[!info] 정보
[!tip] 팁, 힌트
[!success] 성공, 완료
[!question] 질문, FAQ
[!warning] 경고, 주의
[!failure] 실패, 오류
[!danger] 위험
[!bug] 버그
[!example] 예시
[!quote] 인용

중첩 Callout

Markdown 
> [!example] 중첩 예제
> 이것은 외부 callout입니다.
> 
> > [!note]
> > 이것은 내부 callout입니다.
> > 
> > > [!tip]
> > > 더 깊은 중첩도 가능합니다.
