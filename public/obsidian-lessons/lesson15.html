1. 캔버스 소개

무한한 디지털 화이트보드

Obsidian 캔버스는 아이디어를 시각적으로 연결하고 구성할 수 있는 무한한 공간입니다.

무한한 공간

제한 없는 캔버스에서 자유롭게 사고

다양한 카드 타입

텍스트, 노트, 이미지, 웹 페이지

유연한 연결

화살표로 관계와 흐름 표현

시각적 구성

색상, 그룹, 정렬로 체계화

기본 조작법

동작 방법 설명
캔버스 이동 마우스 드래그 / 스페이스 + 드래그 캔버스 공간 탐색
확대/축소 Ctrl/Cmd + 마우스휠 뷰 크기 조절
카드 추가 더블클릭 / 드래그&드롭 새 요소 생성
연결선 그리기 카드 가장자리에서 드래그 관계 생성
다중 선택 Shift + 클릭 / 드래그 선택 여러 요소 선택

# MOC (Map of Contents) 캔버스

1. 주제별 허브 노트를 중앙에 배치
2. 관련 하위 노트들을 주변에 연결
3. 색상으로 카테고리 구분
4. 중요도에 따라 크기 조절

예시:
- 중앙: [[프로젝트 홈]]
- 주변: [[요구사항]], [[설계]], [[개발]], [[테스트]]
- 참고: 외부 링크, 이미지, 문서

6. 실습 프로젝트

🎨 지식 정원 캔버스 만들기

개인 지식 관리 시스템을 시각적으로 표현하는 마스터 캔버스를 만들어보세요.

프로젝트 단계

Phase 1: 구조 설계
  • 주요 관심 분야 식별
  • 각 분야별 색상 지정
  • 중심 허브 생성
Phase 2: 콘텐츠 배치
  • 핵심 노트 카드 추가
  • 관계 화살표 연결
  • 참고 자료 링크
Phase 3: 시각화 개선
  • 그룹별 배경 추가
  • 아이콘과 이미지 활용
  • 레이아웃 최적화
Phase 4: 상호작용 추가
  • 대시보드 노트 연결
  • 작업 추적 영역
  • 학습 경로 표시
