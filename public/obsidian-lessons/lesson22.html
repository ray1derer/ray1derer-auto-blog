제22강: CSS 커스터마이징

CSS를 활용하여 나만의 Obsidian 테마를 만들고 UI를 완벽하게 커스터마이징합니다

고급 75분 CSS

1. CSS 커스터마이징 소개

Obsidian의 CSS 시스템

Obsidian은 강력한 CSS 커스터마이징 시스템을 제공합니다. CSS 스니펫, 커스텀 테마, CSS 변수를 통해 모든 UI 요소를 자유롭게 스타일링할 수 있습니다.

주요 CSS 변수

--background-primary #1e1e1e
--background-secondary #252525
--text-normal #dcddde
--text-muted #999
--interactive-accent #7f6df2
--interactive-hover #6653d3

CSS 적용 방법

  • CSS 스니펫: .obsidian/snippets 폴더에 .css 파일 추가
  • 커스텀 테마: 완전한 테마 파일 생성 및 배포
  • 개발자 도구: 실시간 CSS 수정 및 테스트
  • Style Settings 플러그인: GUI로 CSS 변수 조정

4. 실용적인 CSS 스니펫

인기 CSS 스니펫 갤러리

📝 읽기 전용 라인 너비
읽기 모드에서 최적의 가독성을 위한 라인 너비 제한
.markdown-preview-view { max-width: 750px; margin: 0 auto; padding: 2rem; }
🎨 체크박스 스타일링
작업 상태에 따른 다채로운 체크박스 디자인
input[type=checkbox]:checked { background: #51cf66; border-color: #51cf66; } .task-list-item.is-checked { text-decoration: line-through; opacity: 0.7; }
🏷️ 태그 꾸미기
태그를 알약 모양으로 스타일링
.tag { background: var(--interactive-accent); color: white; padding: 0.2rem 0.6rem; border-radius: 1rem; font-size: 0.85rem; }
📊 테이블 개선
보기 좋은 테이블 스타일링
table { border-collapse: collapse; width: 100%; } th { background: var(--background-secondary); font-weight: 600; } tr:nth-child(even) { background: var(--background-primary-alt); }
💡 콜아웃 커스터마이징
다양한 콜아웃 타입별 스타일
.callout[data-callout="info"] { --callout-color: 66, 153, 225; --callout-icon: info-circle; } .callout[data-callout="warning"] { --callout-color: 236, 201, 75; --callout-icon: alert-triangle; }
🔗 링크 효과
호버 시 부드러운 언더라인 애니메이션
a { position: relative; text-decoration: none; } a::after { content: ''; position: absolute; bottom: -2px; left: 0; width: 0; height: 2px; background: var(--interactive-accent); transition: width 0.3s; } a:hover::after { width: 100%; }

FAQ

CSS 스니펫이 적용되지 않아요
.obsidian/snippets 폴더에 .css 파일을 저장한 후, 설정 > 외관 > CSS 스니펫에서 토글을 켜야 합니다. 파일명에 공백이 없는지도 확인하세요.
!important를 사용해야 하나요?
가급적 피하는 것이 좋습니다. 더 구체적인 선택자를 사용하거나 CSS 변수를 재정의하는 방법을 먼저 시도해보세요. 꼭 필요한 경우에만 사용하세요.
모바일에서 CSS가 다르게 보여요
모바일과 데스크톱의 클래스 구조가 다를 수 있습니다. 미디어 쿼리를 사용하고, 모바일 앱에서 직접 테스트하는 것이 중요합니다.
테마를 커뮤니티에 공유하려면?
GitHub에 리포지토리를 만들고, obsidian-releases 리포지토리에 PR을 제출하면 됩니다. 테마 가이드라인을 준수해야 승인됩니다.
CSS로 기능을 추가할 수 있나요?
CSS는 스타일링만 가능합니다. 새로운 기능을 추가하려면 플러그인을 개발해야 합니다. 다만 ::before, ::after를 활용한 시각적 요소 추가는 가능합니다.
