PKM이란 무엇인가?

Personal Knowledge Management(PKM)는 개인이 정보를 수집, 처리, 저장, 검색, 활용하는 일련의 프로세스와 도구를 의미합니다. 이는 단순한 정보 저장을 넘어 지식의 창조와 혁신을 가능하게 하는 체계입니다.

PKM의 핵심 원칙

연결성

정보는 고립되어서는 가치가 없습니다. 연결을 통해 새로운 인사이트가 생성됩니다.

반복성

지식은 반복적인 검토와 개선을 통해 깊어지고 내재화됩니다.

진화성

지식 시스템은 고정되지 않고 지속적으로 성장하고 변화해야 합니다.

개인화

자신의 사고 방식과 작업 스타일에 맞는 시스템을 구축해야 합니다.

Zettelkasten 방법론

Zettelkasten(메모 상자)은 독일의 사회학자 니클라스 루만이 개발한 노트 작성 시스템으로, 원자적 노트와 연결을 통해 창발적 사고를 가능하게 합니다.

Zettelkasten의 핵심 원칙

1. 원자성 (Atomicity)

하나의 노트는 하나의 아이디어만 담아야 합니다.

2. 자율성 (Autonomy)

각 노트는 독립적으로 이해 가능해야 합니다.

3. 연결성 (Connectivity)

노트 간의 연결을 통해 지식 네트워크를 구축합니다.

UID 생성 시스템

JavaScript 
// Zettelkasten UID 생성기
function generateZettelID() {
    const now = new Date();
    const year = now.getFullYear();
    const month = String(now.getMonth() + 1).padStart(2, '0');
    const day = String(now.getDate()).padStart(2, '0');
    const hour = String(now.getHours()).padStart(2, '0');
    const minute = String(now.getMinutes()).padStart(2, '0');
    
    // 기본 ID: YYYYMMDDHHMM
    const baseID = `${year}${month}${day}${hour}${minute}`;
    
    // 충돌 방지를 위한 suffix
    const suffix = Math.random().toString(36).substr(2, 2);
    
    return `${baseID}${suffix}`;
}

// 계층적 ID 생성
function generateHierarchicalID(parentID, branch) {
    if (!parentID) return '1';
    
    const parts = parentID.split('.');
    const lastPart = parts[parts.length - 1];
    
    if (branch === 'continue') {
        // 1.2 → 1.3
        const num = parseInt(lastPart) + 1;
        parts[parts.length - 1] = num.toString();
    } else if (branch === 'branch') {
        // 1.2 → 1.2.1
        parts.push('1');
    }
    
    return parts.join('.');
}

PARA 방법론

PARA는 Projects, Areas, Resources, Archives의 약자로, Tiago Forte가 개발한 정보 조직 시스템입니다.

Projects

명확한 완료 시점이 있는 활동

  • 논문 작성
  • 앱 개발
  • 강의 준비

Areas

지속적으로 관리해야 하는 책임 영역

  • 건강 관리
  • 재정 관리
  • 가족 관계

Resources

미래에 유용할 수 있는 참고 자료

  • 기술 문서
  • 레시피
  • 인용구

Archives

비활성화된 이전 항목들

  • 완료된 프로젝트
  • 중단된 활동
  • 오래된 자료

PKM 자동화

스마트 노트 분류기

JavaScript 
// 자동 노트 분류 시스템
class SmartNoteCategorizer {
    constructor() {
        this.patterns = {
            project: [
                /deadline|마감|milestone|일정/i,
                /프로젝트|project|개발|development/i
            ],
            area: [
                /관리|management|유지|maintain/i,
                /건강|health|재정|finance/i
            ],
            resource: [
                /참고|reference|문서|document/i,
                /가이드|guide|튜토리얼|tutorial/i
            ],
            zettel: [
                /개념|concept|이론|theory/i,
                /정의|definition|원리|principle/i
            ]
        };
    }
    
    async categorizeNote(note) {
        const content = note.content.toLowerCase();
        const title = note.basename.toLowerCase();
        const text = title + " " + content;
        
        let scores = {
            project: 0,
            area: 0,
            resource: 0,
            zettel: 0
        };
        
        // 패턴 매칭으로 점수 계산
        for (const [category, patterns] of Object.entries(this.patterns)) {
            patterns.forEach(pattern => {
                const matches = text.match(pattern);
                if (matches) {
                    scores[category] += matches.length;
                }
            });
        }
        
        // 메타데이터 기반 가중치
        if (note.frontmatter) {
            if (note.frontmatter.deadline) scores.project += 5;
            if (note.frontmatter.type === 'evergreen') scores.zettel += 5;
            if (note.frontmatter.status === 'ongoing') scores.area += 3;
        }
        
        // 최고 점수 카테고리 반환
        return Object.entries(scores)
            .sort(([,a], [,b]) => b - a)[0][0];
    }
    
    async organizeVault(vault) {
        const uncategorized = vault.getFiles()
            .filter(f => f.parent.path === 'Inbox');
        
        for (const file of uncategorized) {
            const category = await this.categorizeNote(file);
            const targetFolder = this.getCategoryFolder(category);
            
            // 파일 이동
            await vault.rename(
                file,
                `${targetFolder}/${file.name}`
            );
            
            // 카테고리 태그 추가
            await this.addCategoryTag(file, category);
        }
    }
    
    getCategoryFolder(category) {
        const folderMap = {
            project: '1_Projects',
            area: '2_Areas',
            resource: '3_Resources',
            zettel: 'Zettelkasten'
        };
        return folderMap[category] || 'Inbox';
    }
}

```dataviewjs
// PKM 대시보드
const now = moment();
const thirtyDaysAgo = now.subtract(30, 'days');

// 노트 생성 추세
const dailyNotes = {};
for (let i = 0; i < 30; i++) {
    const date = moment().subtract(i, 'days').format('YYYY-MM-DD');
    dailyNotes[date] = 0;
}

dv.pages().forEach(p => {
    const created = moment(p.file.ctime);
    if (created.isAfter(thirtyDaysAgo)) {
        const date = created.format('YYYY-MM-DD');
        if (dailyNotes[date] !== undefined) {
            dailyNotes[date]++;
        }
    }
});

// 차트 데이터 준비
const chartData = {
    type: 'line',
    data: {
        labels: Object.keys(dailyNotes).reverse(),
        datasets: [{
            label: '일일 노트 생성',
            data: Object.values(dailyNotes).reverse(),
            borderColor: '#7c3aed',
            tension: 0.1
        }]
    }
};

// 품질 지표
const qualityMetrics = dv.pages()
    .map(p => ({
        name: p.file.name,
        links: p.file.outlinks.length,
        backlinks: p.file.inlinks.length,
        words: p.file.size,
        tags: p.file.tags.length,
        score: p.file.outlinks.length * 2 + 
               p.file.inlinks.length * 3 + 
               p.file.tags.length
    }))
    .sort(p => p.score, 'desc');

dv.header(2, "📈 지식 성장 지표");

// 성장률 계산
const thisMonth = dv.pages()
    .where(p => moment(p.file.ctime).month() === moment().month()).length;
const lastMonth = dv.pages()
    .where(p => moment(p.file.ctime).month() === moment().month() - 1).length;
const growthRate = ((thisMonth - lastMonth) / lastMonth * 100).toFixed(1);

dv.paragraph(`📊 이번 달 성장률: ${growthRate}%`);
dv.paragraph(`📝 이번 달 생성: ${thisMonth}개`);
dv.paragraph(`🔗 평균 연결: ${(qualityMetrics.map(m => m.links).reduce((a,b) => a+b, 0) / qualityMetrics.length).toFixed(1)}개`);

dv.header(3, "🏆 높은 품질의 노트");
dv.table(["노트", "연결", "역링크", "점수"],
    qualityMetrics.slice(0, 10).map(m => [
        `[[${m.name}]]`,
        m.links,
        m.backlinks,
        m.score
    ])
);
```
