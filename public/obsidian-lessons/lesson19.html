1. 모바일 앱 시작하기

설치와 초기 설정

iOS (iPhone/iPad)

  1. App Store에서 "Obsidian" 검색
  2. 무료 다운로드 및 설치
  3. iOS 15.4 이상 필요
  4. iPad는 별도 최적화 UI 제공
iOS 특별 기능
  • Files 앱 통합
  • Shortcuts 지원
  • Share Sheet 확장
  • Split View (iPad)

Android

  1. Google Play Store에서 "Obsidian" 검색
  2. 무료 다운로드 및 설치
  3. Android 7.0 이상 필요
  4. 태블릿 레이아웃 지원
Android 특별 기능
  • 외부 저장소 접근
  • 위젯 지원
  • Intent 공유
  • Termux 연동 가능

첫 실행 설정

1

볼트 선택

  • 새 볼트 생성
  • 기존 볼트 열기
  • 클라우드 폴더에서 열기
2

동기화 방법 선택

  • Obsidian Sync (권장)
  • iCloud (iOS)
  • Google Drive
  • Dropbox
3

기본 설정

  • 테마 선택
  • 글꼴 크기
  • 편집 모드

3. 동기화 설정

동기화 방법 비교

방법 장점 단점 권장 사용
Obsidian Sync
  • 실시간 동기화
  • 버전 관리
  • 엔드투엔드 암호화
  • 충돌 해결
  • 유료 ($8/월)
  • 5GB 제한
 전문가/헤비유저
iCloud
  • iOS 통합
  • 자동 백업
  • 무료 5GB
  • iOS 전용
  • 가끔 동기화 지연
 Apple 생태계 사용자
클라우드 스토리지
  • 무료 옵션
  • 크로스 플랫폼
  • 대용량 지원
  • 수동 설정
  • 충돌 가능성
  • 속도 느림
 가벼운 사용자

Obsidian Sync 설정

1. 설정 → Sync → 로그인
2. 원격 볼트 생성/선택
3. 동기화 옵션 설정:
   ✓ 설정 동기화
   ✓ 플러그인 동기화
   ✓ 테마 동기화
   ✓ 이미지 동기화
4. 제외 폴더 설정 (선택사항)
5. 동기화 시작

동기화 문제 해결

🔴 충돌 발생

해결책:

  • 충돌 파일 수동 병합
  • 버전 히스토리 확인
  • 백업 파일 활용

🟡 동기화 지연

해결책:

  • 네트워크 연결 확인
  • 캐시 정리
  • 앱 재시작
