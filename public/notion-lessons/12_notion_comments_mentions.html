Notion 완전 정복: 일상 속 문제 해결부터 시작하는 스마트 워크스페이스 구축

12강: 댓글과 멘션으로 소통하기 (협업 기능 심화)

강의 목표: 이번 강의에서는 Notion의 '댓글'과 '@ 멘션' 기능을 활용하여 팀원들과 효율적으로 소통하고 협업하는 방법을 배웁니다. 페이지나 특정 블록에 대한 피드백을 주고받고, 필요한 사람에게 알림을 보내는 노하우를 익혀 보세요.

12.1. 댓글 (Comments): 페이지와 블록에 대한 피드백 주고받기

Notion의 댓글 기능은 페이지 전체 또는 특정 블록에 대한 의견, 질문, 피드백을 남길 수 있게 해줍니다. 이는 문서 기반의 협업에서 매우 중요한 기능입니다.

  1. 페이지에 댓글 달기:
    • 페이지 우측 상단의 '댓글' 아이콘을 클릭하거나, 페이지 제목 아래에 있는 '댓글 추가' 영역을 클릭합니다.

    스크린샷 가이드: 페이지에 댓글 추가

    • 목적: 페이지 전체에 대한 댓글을 추가하는 방법을 보여줍니다.
    • 가이드: 페이지 우측 상단의 '댓글' 아이콘 또는 페이지 제목 아래의 '댓글 추가' 영역을 클릭했을 때 나타나는 댓글 입력 창을 캡처합니다.
    • 이미지 파일명: 12-1_add_page_comment.png
  2. 특정 블록에 댓글 달기:
    • 댓글을 달고 싶은 블록 위에 마우스를 올리면 왼쪽에 ⋮⋮ 아이콘이 나타납니다. 이 아이콘을 클릭한 후 '댓글' 또는 'Comment'를 선택합니다.

    스크린샷 가이드: 특정 블록에 댓글 추가

    • 목적: 특정 블록에 대한 댓글을 추가하는 방법을 보여줍니다.
    • 가이드: 블록 메뉴에서 '댓글' 옵션을 클릭했을 때 나타나는 댓글 입력 창을 캡처합니다.
    • 이미지 파일명: 12-2_add_block_comment.png
  3. 댓글 확인 및 관리:
    • 댓글이 달린 페이지나 블록 옆에는 댓글 아이콘과 개수가 표시됩니다. 클릭하면 댓글 내용을 확인하고 답글을 달 수 있습니다.

    스크린샷 가이드: 댓글 확인 및 답글 달기

    • 목적: 달린 댓글을 확인하고 답글을 다는 모습을 보여줍니다.
    • 가이드: 댓글 아이콘을 클릭했을 때 나타나는 댓글 목록과 답글 입력 창을 캡처합니다.
    • 이미지 파일명: 12-3_view_reply_comments.png

12.2. @ 멘션: 필요한 사람에게 알림 보내기

@ 멘션 기능은 특정 팀원이나 페이지, 날짜를 언급하여 알림을 보내거나 정보를 연결할 때 사용합니다.

  1. 팀원 멘션하기:
    • 댓글이나 페이지 내용 입력 중 @를 입력한 후, 언급하고 싶은 팀원의 이름을 입력합니다. Notion이 자동으로 팀원 목록을 보여줍니다.

    스크린샷 가이드: 팀원 멘션

    • 목적: @를 사용하여 팀원을 멘션하는 과정을 보여줍니다.
    • 가이드: @ 입력 후 팀원 이름이 나타나는 자동 완성 목록을 캡처합니다.
    • 이미지 파일명: 12-4_mention_teammate.png
  2. 페이지 멘션하기:
    • @를 입력한 후, 연결하고 싶은 Notion 페이지의 이름을 입력합니다. 해당 페이지로 바로 이동할 수 있는 링크가 생성됩니다.

    스크린샷 가이드: 페이지 멘션

    • 목적: @를 사용하여 페이지를 멘션하는 과정을 보여줍니다.
    • 가이드: @ 입력 후 페이지 이름이 나타나는 자동 완성 목록을 캡처합니다.
    • 이미지 파일명: 12-5_mention_page.png
  3. 날짜 멘션하기:
    • @를 입력한 후, 특정 날짜(예: 오늘, 내일, 다음 주 월요일 또는 특정 날짜)를 입력합니다. 해당 날짜에 대한 알림을 설정할 수 있습니다.

    스크린샷 가이드: 날짜 멘션

    • 목적: @를 사용하여 날짜를 멘션하고 알림을 설정하는 과정을 보여줍니다.
    • 가이드: @ 입력 후 날짜가 나타나는 자동 완성 목록과 알림 설정 옵션을 캡처합니다.
    • 이미지 파일명: 12-6_mention_date.png

12.3. 알림 (Updates): 모든 소통을 한눈에 확인하기

Notion의 '알림(Updates)' 메뉴는 여러분과 관련된 모든 댓글, 멘션, 페이지 변경 사항을 한곳에서 확인할 수 있게 해줍니다.

  1. 알림 메뉴 접속:
    • Notion 워크스페이스의 왼쪽 사이드바 상단에 있는 '알림' 또는 'Updates'를 클릭합니다.

    스크린샷 가이드: 알림 메뉴 접속

    • 목적: 알림 메뉴에 접속하는 방법을 보여줍니다.
    • 가이드: 사이드바에서 '알림' 메뉴를 클릭하는 모습을 캡처합니다.
    • 이미지 파일명: 12-7_access_updates_menu.png
  2. 알림 확인 및 처리:
    • 알림 메뉴에서는 여러분이 멘션된 내용, 댓글, 팔로우하는 페이지의 변경 사항 등을 확인할 수 있습니다.
    • 알림을 클릭하면 해당 페이지나 블록으로 바로 이동할 수 있습니다.

    스크린샷 가이드: 알림 목록 확인

    • 목적: 알림 메뉴에 표시되는 알림 목록을 보여줍니다.
    • 가이드: 알림 메뉴에 여러 개의 알림이 표시된 화면을 캡처합니다.
    • 이미지 파일명: 12-8_view_notification_list.png

12.4. 협업 시나리오: 프로젝트 피드백 주고받기

Notion의 댓글과 멘션 기능을 활용하여 프로젝트 피드백을 효율적으로 주고받는 시나리오를 살펴보겠습니다.

  • 상황: 팀 프로젝트 기획 페이지를 작성 중이며, 특정 부분에 대해 팀원의 피드백이 필요합니다.
  • 활용:
    1. 기획 페이지의 특정 문단에 마우스를 올리고 ⋮⋮ 아이콘을 클릭하여 '댓글'을 추가합니다.
    2. 댓글 내용에 "이 부분에 대한 @김노션 님의 의견이 필요합니다. 특히 마케팅 전략에 대해 검토 부탁드립니다."라고 입력합니다.
    3. 김노션님은 알림을 통해 해당 댓글을 확인하고, 페이지로 이동하여 댓글에 답글을 달거나, 직접 페이지 내용을 수정하고 "수정 완료 @박노션"이라고 멘션하여 알림을 보냅니다.

이처럼 댓글과 멘션은 Notion 내에서 실시간으로 소통하고, 피드백을 주고받으며, 협업의 효율성을 높이는 데 큰 도움을 줍니다.

강의 요약:

이번 12강에서는 Notion의 '댓글'과 '@ 멘션' 기능을 활용하여 팀원들과 효율적으로 소통하고 협업하는 방법을 배웠습니다. 페이지나 특정 블록에 대한 피드백을 주고받고, 필요한 사람에게 알림을 보내며, '알림' 메뉴를 통해 모든 소통 내역을 한눈에 확인하는 방법을 익혔습니다.
