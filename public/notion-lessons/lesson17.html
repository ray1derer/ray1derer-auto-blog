17강: 워크플로우 자동화 - 데이터베이스 템플릿

반복적인 작업을 줄이고 일관된 데이터 입력을 위해 노션 데이터베이스 템플릿을 활용하는 방법을 배웁니다. 데이터베이스 템플릿은 새로운 항목을 생성할 때 미리 정의된 속성 값과 페이지 콘텐츠를 자동으로 채워 넣어 워크플로우를 효율적으로 자동화하는 데 도움을 줍니다.

학습 목표

  • 데이터베이스 템플릿을 만들어 속성과 페이지 콘텐츠를 미리 채웁니다.
  • 데이터베이스의 기본 템플릿을 설정합니다.

주요 개념

  • 데이터베이스 템플릿 (Database Template): 데이터베이스에 새로운 항목(페이지)을 추가할 때 사용할 수 있는 미리 정의된 구조와 콘텐츠입니다. 속성 값, 텍스트 블록, 하위 페이지 등을 포함할 수 있습니다.
  • 기본 템플릿 (Default Template): 특정 데이터베이스에 대해 기본 템플릿을 설정하면, 새로운 항목을 생성할 때마다 해당 템플릿이 자동으로 적용됩니다.

전문가 팁

템플릿 내에 연결된 데이터베이스 보기를 포함하는 방법을 학습합니다. 예를 들어, '새 프로젝트' 템플릿은 자동으로 해당 프로젝트의 업무만 필터링해서 보여주는 보기를 포함할 수 있습니다. 이는 프로젝트 페이지를 열었을 때 해당 프로젝트와 관련된 모든 업무를 한눈에 볼 수 있게 하여 효율성을 극대화합니다.

실습 과제

15강에서 만든 '프로젝트' 데이터베이스에 '새 클라이언트 프로젝트' 템플릿을 만듭니다. 이 템플릿은 다음을 포함하도록 설정해 보세요:

  • '상태' 속성을 '시작 전'으로 자동 설정합니다.
  • '프로젝트 개요', '주요 연락처', '관련 업무' 등의 H2 제목을 포함하는 페이지 구조를 미리 갖추도록 합니다.
  • '관련 업무' 섹션 아래에 '업무' 데이터베이스의 연결된 보기를 추가하고, 현재 프로젝트에 해당하는 업무만 필터링되도록 설정합니다.
