16강: 데이터 집계하기 - 롤업 속성

관계형 속성을 통해 데이터베이스를 연결했다면, 이제 연결된 데이터베이스의 정보를 가져와 요약하고 계산할 수 있습니다. '롤업(Rollup)' 속성은 이러한 데이터 집계 기능을 제공하여, 복잡한 정보를 한눈에 파악하고 의사결정을 돕는 강력한 도구입니다. 이 강의에서는 롤업 속성을 사용하여 연결된 데이터의 합계, 평균, 비율 등을 계산하는 방법을 배웁니다.

학습 목표

  • 롤업 속성을 사용하여 관계형으로 연결된 데이터베이스에서 데이터를 가져옵니다.
  • 롤업 데이터에 대한 계산(예: 합계, 평균, 체크된 항목 비율)을 수행합니다.

주요 개념

  • 롤업 속성 (Rollup Property): 관계형 속성을 통해 연결된 다른 데이터베이스의 속성 값을 가져와 표시하거나, 해당 값들을 기반으로 다양한 계산을 수행할 수 있는 속성입니다.
  • 집계 (Aggregation): 롤업 속성에서 수행할 수 있는 계산 유형을 의미합니다. 예를 들어, 연결된 항목들의 속성 값에 대해 '합계', '평균', '최소', '최대', '개수', '체크된 항목 비율' 등을 계산할 수 있습니다.
  • 계산 옵션: 롤업 속성을 설정할 때, 어떤 관계형 속성을 참조할지, 어떤 속성 값을 가져올지, 그리고 어떤 방식으로 집계할지를 선택하는 옵션입니다.

실습 과제

15강에서 만든 '프로젝트' 데이터베이스에 롤업 속성을 추가합니다. 이 롤업이 연결된 '업무' 데이터베이스를 참조하여 다음을 설정해 보세요:

  • '업무' 데이터베이스의 '상태' 속성(예: 할 일, 진행 중, 완료)을 가져옵니다.
  • 가져온 '상태' 속성 중 '완료'된 항목의 '체크된 항목 비율'을 계산하여 실시간 진행률 막대를 표시하도록 설정합니다. (예: 2/3 완료 또는 66%)
