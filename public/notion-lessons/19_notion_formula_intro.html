19강: 수식(Formula) 기초 (1) - 기본 개념과 간단한 함수

강의 목표: 이번 강의에서는 Notion 데이터베이스를 한 단계 더 업그레이드해 줄 '수식(Formula)' 기능의 기초를 배웁니다. 수식의 기본 개념을 이해하고, 간단한 연산자와 함수를 사용하여 텍스트를 조합하거나 날짜를 계산하는 등 데이터 자동화의 첫걸음을 내딛습니다.

19.1. 수식(Formula)이란? (Notion의 Excel)

수식은 데이터베이스의 다른 속성 값을 기반으로 새로운 값을 계산하거나, 텍스트를 조작하고, 특정 조건에 따라 다른 결과를 보여주는 강력한 기능입니다. 마치 Notion 속성 안에 작은 Excel이 들어있는 것과 같습니다.

  • 자동 계산: '가격'과 '수량' 속성으로 '총액'을 자동 계산
  • 상태 표시: 마감일까지 남은 날짜(D-day)를 계산하여 표시
  • 텍스트 조합: '성'과 '이름' 속성을 합쳐 '전체 이름' 표시

스크린샷 가이드: 수식 속성 추가

  • 목적: 데이터베이스에 수식 속성을 추가하는 방법을 보여줍니다.
  • 가이드: 데이터베이스에서 '+' 버튼을 눌러 새 속성을 추가할 때, 속성 유형 목록에서 '수식'을 선택하는 화면을 캡처합니다.
  • 이미지 파일명: 19-1_add_formula_property.png

19.2. 기본 연산자와 텍스트 합치기

수식의 가장 기본은 사칙연산과 텍스트 조합입니다.

  • 숫자 연산: +, -, *, / 등 일반적인 연산자를 사용할 수 있습니다.
    prop("가격") * prop("수량")
  • 텍스트 조합 (concat): + 연산자는 텍스트를 합치는 데도 사용됩니다.
    "이름: " + prop("담당자")

스크린샷 가이드: 간단한 수식 예제

  • 목적: 수식 편집기에서 간단한 텍스트 조합 수식을 작성하는 모습을 보여줍니다.
  • 가이드: '성'과 '이름' 속성을 가진 데이터베이스에서, prop("성") + prop("이름") 이라는 수식을 작성하여 '전체이름'을 만드는 화면을 캡처합니다.
  • 이미지 파일명: 19-2_formula_text_concat.png

19.3. 날짜 계산하기: dateBetween()

dateBetween() 함수는 두 날짜 사이의 간격을 계산할 때 사용합니다. 프로젝트 기간이나 D-day를 계산하는 데 매우 유용합니다.

구문: dateBetween(날짜1, 날짜2, "단위")

예시 (D-day 계산): 마감일까지 남은 날짜 계산하기

dateBetween(prop("마감일"), now(), "days")

now()는 현재 시간을 나타내는 함수입니다.

스크린샷 가이드: dateBetween 함수로 D-day 계산

  • 목적: dateBetween 함수를 사용하여 D-day를 계산하는 수식을 보여줍니다.
  • 가이드: '할 일' 데이터베이스에 '마감일' 속성이 있고, 수식으로 D-day가 계산되어 숫자로 표시되는 열을 캡처합니다.
  • 이미지 파일명: 19-3_formula_datebetween_dday.png

19.4. 간단한 조건문으로 자동화하기: if()

if() 함수는 특정 조건이 참인지 거짓인지에 따라 다른 결과를 반환하게 만드는, 자동화의 핵심 함수입니다.

구문: if(조건, 참일 때 결과, 거짓일 때 결과)

예시 (상태 자동 표시): 체크박스가 선택되면 "완료", 아니면 "미완료" 표시하기

if(prop("완료 여부"), "✅ 완료", "🏃 진행 중")

체크박스 속성은 그 자체로 참(true)/거짓(false) 값을 가집니다.

스크린샷 가이드: if 함수로 상태 표시 자동화

  • 목적: if 함수를 사용하여 다른 속성 값에 따라 결과가 바뀌는 것을 보여줍니다.
  • 가이드: '완료 여부' 체크박스 속성과, 체크박스 상태에 따라 "✅ 완료" 또는 "🏃 진행 중"으로 자동으로 바뀌는 수식 속성 열을 캡처합니다.
  • 이미지 파일명: 19-4_formula_if_statement.png

강의 요약:

이번 19강에서는 수식(Formula)의 기본 개념을 배우고, 기본 연산자, 텍스트를 합치는 +, 날짜 간격을 계산하는 dateBetween(), 조건에 따라 다른 값을 보여주는 if() 함수를 익혔습니다. 이제 단순한 데이터 입력을 넘어, 데이터 간의 관계를 활용하여 정보를 자동으로 가공하는 첫발을 내디뎠습니다.
