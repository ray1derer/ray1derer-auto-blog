Notion 완전 정복: 일상 속 문제 해결부터 시작하는 스마트 워크스페이스 구축

7강: 페이지를 예쁘게 꾸미는 법 (아이콘, 커버, 색상, 글꼴, 레이아웃)

강의 목표: 이번 강의에서는 Notion 페이지를 시각적으로 매력 있게 꾸미는 다양한 방법을 배웁니다. 아이콘, 커버 이미지, 텍스트 색상, 글꼴, 그리고 페이지 레이아웃을 활용하여 나만의 개성 있는 Notion 페이지를 만들어 보세요.

7.1. 페이지 아이콘 추가 및 변경 (페이지의 첫인상)

  1. 아이콘 추가:
    • 새 페이지를 만들거나 기존 페이지의 제목 위에 마우스를 올리면 '아이콘 추가' 버튼이 나타납니다. 클릭합니다.

    스크린샷 가이드: 페이지 아이콘 추가 버튼

    • 목적: 페이지 제목 위에 마우스를 올렸을 때 나타나는 '아이콘 추가' 버튼을 보여줍니다.
    • 가이드: Notion 페이지의 제목 위에 마우스를 올린 상태를 캡처합니다.
    • 이미지 파일명: 7-1_add_page_icon.png
  2. 아이콘 유형 선택:
    • 이모지(Emoji): 가장 간편하게 사용할 수 있는 아이콘입니다. 다양한 이모지 중에서 페이지 내용에 맞는 것을 선택합니다.
    • 아이콘(Icon): Notion이 제공하는 기본 아이콘 세트입니다. 깔끔하고 통일감 있는 디자인을 선호할 때 좋습니다.
    • 커스텀(Custom): 직접 만든 이미지 파일(PNG, JPG, GIF 등)을 업로드하여 아이콘으로 사용할 수 있습니다.
    • 링크(Link): 웹에 있는 이미지의 URL을 붙여넣어 아이콘으로 사용할 수 있습니다.

    스크린샷 가이드: 아이콘 유형 선택 메뉴

    • 목적: 아이콘 추가 버튼 클릭 시 나타나는 이모지, 아이콘, 커스텀, 링크 옵션을 보여줍니다.
    • 가이드: 아이콘 추가 버튼 클릭 후 나타나는 팝업 메뉴를 캡처합니다.
    • 이미지 파일명: 7-2_select_icon_type.png

7.2. 페이지 커버 이미지 추가 및 변경 (페이지의 분위기)

  1. 커버 추가:
    • 페이지 제목 위에 마우스를 올리면 '커버 추가' 버튼이 나타납니다. 클릭합니다.

    스크린샷 가이드: 페이지 커버 추가 버튼

    • 목적: 페이지 제목 위에 마우스를 올렸을 때 나타나는 '커버 추가' 버튼을 보여줍니다.
    • 가이드: Notion 페이지의 제목 위에 마우스를 올린 상태를 캡처합니다.
    • 이미지 파일명: 7-3_add_page_cover.png
  2. 커버 이미지 유형 선택:
    • 갤러리(Gallery): Notion이 제공하는 기본 이미지 갤러리입니다.
    • 업로드(Upload): 직접 만든 이미지 파일을 업로드합니다.
    • 링크(Link): 웹에 있는 이미지의 URL을 붙여넣어 사용합니다.
    • Unsplash: 고품질의 무료 이미지를 검색하여 사용할 수 있습니다.

    스크린샷 가이드: 커버 이미지 유형 선택 메뉴

    • 목적: 커버 추가 버튼 클릭 시 나타나는 갤러리, 업로드, 링크, Unsplash 옵션을 보여줍니다.
    • 가이드: 커버 추가 버튼 클릭 후 나타나는 팝업 메뉴를 캡처합니다.
    • 이미지 파일명: 7-4_select_cover_type.png
  3. 커버 위치 조절:
    • 커버 이미지를 추가한 후, '위치 변경' 버튼을 클릭하여 이미지의 보이는 부분을 조절할 수 있습니다.

    스크린샷 가이드: 커버 위치 조절

    • 목적: 커버 이미지의 보이는 부분을 조절하는 모습을 보여줍니다.
    • 가이드: '위치 변경' 버튼 클릭 후 커버 이미지를 드래그하여 위치를 조절하는 모습을 캡처합니다.
    • 이미지 파일명: 7-5_adjust_cover_position.png

7.3. 텍스트 색상 및 배경색 변경하기 (내용 강조)

  1. 텍스트 선택:
    • 색상을 변경하고 싶은 텍스트 블록을 마우스로 드래그하여 선택합니다.

    스크린샷 가이드: 텍스트 선택

    • 목적: 색상을 변경할 텍스트를 선택하는 모습을 보여줍니다.
    • 가이드: 텍스트 블록의 일부 또는 전체를 드래그하여 선택한 상태를 캡처합니다.
    • 이미지 파일명: 7-6_select_text_for_color.png
  2. 색상 팔레트 열기:
    • 텍스트 선택 시 나타나는 서식 메뉴에서 'A' 아이콘(색상)을 클릭합니다.

    스크린샷 가이드: 색상 팔레트 열기

    • 목적: 텍스트 서식 메뉴에서 색상 팔레트를 여는 모습을 보여줍니다.
    • 가이드: 'A' 아이콘을 클릭했을 때 나타나는 색상 팔레트를 캡처합니다.
    • 이미지 파일명: 7-7_open_color_palette.png
  3. 색상 선택:
    • 원하는 텍스트 색상이나 배경색을 선택합니다.

    스크린샷 가이드: 색상 선택 및 적용

    • 목적: 색상 팔레트에서 색상을 선택하고 텍스트에 적용된 모습을 보여줍니다.
    • 가이드: 특정 색상을 클릭하여 텍스트에 적용된 후의 화면을 캡처합니다.
    • 이미지 파일명: 7-8_apply_text_color.png

7.4. 글꼴 스타일 변경 (페이지의 분위기 전환)

  1. 글꼴 메뉴 열기:
    • 페이지 우측 상단의 '...' 메뉴를 클릭합니다.

    스크린샷 가이드: 페이지 설정 메뉴 열기

    • 목적: 페이지 우측 상단의 '...' 메뉴를 클릭하는 모습을 보여줍니다.
    • 가이드: Notion 페이지 우측 상단의 '...' 아이콘을 클릭했을 때 나타나는 드롭다운 메뉴를 캡처합니다.
    • 이미지 파일명: 7-9_open_page_settings.png
  2. 글꼴 선택:
    • 드롭다운 메뉴에서 '글꼴' 또는 'Font' 옵션을 찾아 원하는 글꼴 스타일(기본, 세리프, 모노)을 선택합니다.

    스크린샷 가이드: 글꼴 선택

    • 목적: 글꼴 선택 옵션과 각 글꼴 스타일이 적용된 모습을 보여줍니다.
    • 가이드: '글꼴' 옵션 아래에 나타나는 '기본', '세리프', '모노' 선택지를 캡처합니다.
    • 이미지 파일명: 7-10_select_font_style.png

7.5. 페이지 레이아웃 변경 (열 나누기, 전체 너비)

  1. 열 나누기 (Columns):
    • 블록을 드래그하여 다른 블록 옆으로 이동시키면 파란색 세로 가이드라인이 나타납니다. 이때 마우스를 놓으면 블록들이 여러 열로 나뉘어 배치됩니다.

    스크린샷 가이드: 블록을 이용한 열 나누기

    • 목적: 두 개의 텍스트 블록을 드래그하여 2열로 나누는 모습을 보여줍니다.
    • 가이드: 하나의 블록을 다른 블록 옆으로 드래그했을 때 파란색 세로 가이드라인이 나타난 화면을 캡처합니다.
    • 이미지 파일명: 7-11_create_columns.png
  2. 전체 너비 (Full Width):
    • 페이지 우측 상단의 '...' 메뉴를 클릭한 후 '전체 너비' 또는 'Full width'를 활성화하면 페이지의 내용이 화면 전체를 사용하도록 확장됩니다. 넓은 화면에서 많은 정보를 볼 때 유용합니다.

    스크린샷 가이드: 전체 너비 설정

    • 목적: 페이지 설정 메뉴에서 '전체 너비' 옵션을 활성화하는 모습을 보여줍니다.
    • 가이드: '...' 메뉴에서 '전체 너비' 체크박스가 활성화된 화면을 캡처합니다.
    • 이미지 파일명: 7-12_full_width_setting.png

강의 요약:

이번 7강에서는 Notion 페이지를 시각적으로 매력 있게 꾸미는 다양한 방법을 배웠습니다. 아이콘과 커버 이미지로 페이지의 첫인상과 분위기를 만들고, 텍스트 색상과 글꼴로 내용을 강조하며, 열 나누기와 전체 너비 설정을 통해 페이지 레이아웃을 효율적으로 구성하는 방법을 익혔습니다. 이제 여러분은 Notion 페이지를 더욱 개성 있고 보기 좋게 만들 수 있습니다.
