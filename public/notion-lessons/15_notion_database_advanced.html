Notion 완전 정복: 일상 속 문제 해결부터 시작하는 스마트 워크스페이스 구축

15강: 데이터베이스 고급 활용 (필터, 정렬, 보기 링크, 템플릿)

강의 목표: 이번 강의에서는 데이터베이스에 쌓인 정보를 효과적으로 관리하는 고급 기능을 배웁니다. 필터와 정렬을 통해 원하는 데이터만 골라보고, 데이터베이스 보기 링크와 템플릿을 활용하여 워크플로우를 자동화하는 방법을 익힙니다.

15.1. 필터(Filter): 원하는 데이터만 골라보기

필터는 특정 조건에 맞는 데이터만 화면에 표시하는 기능입니다. 데이터가 많아질수록 원하는 정보를 빠르게 찾는 데 필수적입니다.

  • 사용법: 데이터베이스 오른쪽 상단의 '필터' 버튼을 클릭하고, 기준이 될 속성과 조건을 설정합니다. (예: '상태'가 '진행 중'인 항목만 보기)
  • 복합 조건: '그리고' 또는 '또는' 조건을 추가하여 여러 개의 필터를 동시에 적용할 수 있습니다. (예: '상태'가 '진행 중'이고 '담당자'가 '나'인 항목)

스크린샷 가이드: 필터 설정 화면

  • 목적: 필터 기능을 사용하는 방법을 보여줍니다.
  • 가이드: '독서 기록장' 데이터베이스에서 '상태'가 '완독'인 책만 필터링하는 화면을 캡처합니다. 필터 설정 메뉴가 열려 있어야 합니다.
  • 이미지 파일명: 15-1_database_filter.png

15.2. 정렬(Sort): 데이터 순서 마음대로 바꾸기

정렬은 특정 속성을 기준으로 데이터의 순서를 재배열하는 기능입니다.

  • 사용법: 데이터베이스 오른쪽 상단의 '정렬' 버튼을 클릭하고, 기준이 될 속성과 정렬 방식(오름차순/내림차순)을 선택합니다.
  • 다중 정렬: 여러 정렬 기준을 추가할 수 있습니다. 첫 번째 기준으로 정렬 후, 같은 값 내에서 두 번째 기준으로 다시 정렬합니다. (예: '완독일' 내림차순으로 정렬 후, '평점' 내림차순으로 정렬)

스크린샷 가이드: 정렬 설정 화면

  • 목적: 정렬 기능을 사용하는 방법을 보여줍니다.
  • 가이드: '독서 기록장' 데이터베이스를 '평점' 내림차순(별이 많은 순)으로 정렬하는 화면을 캡처합니다. 정렬 설정 메뉴가 열려 있어야 합니다.
  • 이미지 파일명: 15-2_database_sort.png

15.3. 데이터베이스 보기 링크 (Linked View of Database)

데이터베이스 보기 링크는 원본 데이터베이스의 '복사본'이 아닌, 실시간으로 동기화되는 '바로가기'를 만드는 기능입니다. 하나의 원본 데이터를 여러 페이지에서 다양한 필터와 정렬을 적용하여 활용할 수 있습니다.

  1. 생성 방법: 원하는 페이지에서 /데이터베이스 보기 링크를 입력하고, 연결할 원본 데이터베이스를 선택합니다.
  2. 활용 예시:
    • '오늘 할 일' 페이지에 전체 할 일 데이터베이스를 링크하고, '마감일'이 '오늘'인 항목만 필터링하여 보여주기
    • 프로젝트 페이지에 전체 회의록 데이터베이스를 링크하고, '관련 프로젝트'가 '현재 프로젝트'인 회의록만 필터링하여 보여주기

스크린샷 가이드: 데이터베이스 보기 링크 생성

  • 목적: 다른 페이지에 원본 데이터베이스의 링크된 보기를 삽입하는 모습을 보여줍니다.
  • 가이드: 새로운 페이지에서 '오늘의 독서'라는 제목 아래에 '독서 기록장' 데이터베이스의 링크된 보기를 생성하고, '상태'가 '읽는 중'인 책만 필터링한 화면을 캡처합니다.
  • 이미지 파일명: 15-3_linked_database_view.png

15.4. 데이터베이스 템플릿: 반복 작업 자동화

데이터베이스 템플릿을 사용하면 데이터베이스에 새 항목(페이지)을 추가할 때마다 미리 설정된 양식을 자동으로 불러올 수 있습니다. 회의록, 프로젝트 기획서, 버그 리포트 등 반복적인 구조를 가진 페이지를 만들 때 매우 유용합니다.

  1. 템플릿 만들기: 데이터베이스 오른쪽 상단의 '새로 만들기' 버튼 옆 드롭다운(▼)을 클릭하고 '새 템플릿'을 선택합니다.
  2. 템플릿 편집: 템플릿 페이지의 제목, 속성 기본값, 페이지 본문 내용을 미리 작성합니다. (예: 회의록 템플릿에 '회의 안건', '결정 사항' 등의 헤더를 미리 넣어두기)
  3. 템플릿 사용: '새로 만들기'를 클릭하거나 템플릿 버튼을 직접 클릭하면, 미리 만들어 둔 양식이 적용된 새 페이지가 생성됩니다.

스크린샷 가이드: 데이터베이스 템플릿 편집 화면

  • 목적: 데이터베이스 템플릿을 만들고 편집하는 과정을 보여줍니다.
  • 가이드: '회의록' 데이터베이스에서 '주간 회의 템플릿'을 만드는 화면을 캡처합니다. 페이지 본문에 '참석자', '안건', '논의 내용' 등의 기본 구조가 작성되어 있는 모습이면 좋습니다.
  • 이미지 파일명: 15-4_database_template.png

강의 요약:

이번 15강에서는 필터와 정렬로 데이터를 효과적으로 제어하고, 데이터베이스 보기 링크로 정보를 재활용하며, 데이터베이스 템플릿으로 반복 작업을 자동화하는 방법을 배웠습니다. 이 기능들을 활용하면 Notion 데이터베이스를 훨씬 더 체계적이고 효율적으로 관리할 수 있습니다.
