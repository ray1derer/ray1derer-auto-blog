Notion 완전 정복: 일상 속 문제 해결부터 시작하는 스마트 워크스페이스 구축

8강: 다양한 블록 활용하기 (비디오, 오디오, 웹 북마크, 코드 블록, 파일)

강의 목표: 이번 강의에서는 Notion 페이지를 더욱 풍부하고 다채롭게 만들 수 있는 다양한 블록들을 살펴봅니다. 비디오, 오디오, 웹 북마크, 코드 블록, 파일 블록을 활용하여 정보를 더욱 효과적으로 정리하고 공유하는 방법을 배웁니다.

8.1. 비디오 블록: 영상 콘텐츠를 페이지에 담기

  1. 비디오 블록 추가:
    • 새로운 블록을 만들 위치에서 /를 입력하고 비디오 또는 video를 입력하여 '비디오' 블록을 선택합니다.

    스크린샷 가이드: 슬래시 명령어로 비디오 블록 추가

    • 목적: 슬래시 명령어를 사용하여 비디오 블록을 추가하는 과정을 보여줍니다.
    • 가이드: / 입력 후 '비디오'를 검색하여 선택하는 모습을 캡처합니다.
    • 이미지 파일명: 8-1_add_video_block.png
  2. 비디오 삽입 방법:
    • 업로드: 컴퓨터에 있는 비디오 파일을 직접 업로드합니다.
    • 링크: 유튜브, Vimeo 등 웹에 있는 비디오의 URL을 붙여넣어 삽입합니다.

    스크린샷 가이드: 비디오 삽입 옵션

    • 목적: 비디오 블록 생성 후 나타나는 비디오 삽입 옵션(업로드, 링크)을 보여줍니다.
    • 가이드: 비디오 블록을 추가한 후 나타나는 '업로드', '링크 삽입' 버튼이 보이는 화면을 캡처합니다.
    • 이미지 파일명: 8-2_video_insert_options.png
  3. 비디오 재생 및 크기 조절:
    • 삽입된 비디오는 Notion 페이지 내에서 바로 재생할 수 있습니다.
    • 이미지 블록과 마찬가지로 비디오 블록도 크기를 조절하고 페이지 내에서 위치를 변경할 수 있습니다.

    스크린샷 가이드: 비디오 재생 및 크기 조절

    • 목적: 삽입된 비디오가 페이지 내에서 재생되는 모습과 크기를 조절하는 모습을 보여줍니다.
    • 가이드: 비디오가 재생 중인 화면과, 비디오 블록의 모서리를 드래그하여 크기를 조절하는 모습을 캡처합니다.
    • 이미지 파일명: 8-3_video_playback_resize.png

  1. 오디오 블록 추가:

8.3. 웹 북마크 블록: 웹 페이지를 깔끔하게 정리하기

단순히 URL을 붙여넣는 것보다 '웹 북마크' 블록을 사용하면 웹 페이지의 제목, 설명, 미리보기 이미지가 함께 표시되어 훨씬 깔끔하게 정리할 수 있습니다.

  1. 웹 북마크 블록 추가:
    • 새로운 블록을 만들 위치에서 /를 입력하고 웹 북마크 또는 web bookmark를 입력하여 '웹 북마크' 블록을 선택합니다.

    스크린샷 가이드: 슬래시 명령어로 웹 북마크 블록 추가

    • 목적: 슬래시 명령어를 사용하여 웹 북마크 블록을 추가하는 과정을 보여줍니다.
    • 가이드: / 입력 후 '웹 북마크'를 검색하여 선택하는 모습을 캡처합니다.
    • 이미지 파일명: 8-6_add_web_bookmark_block.png
  2. URL 붙여넣기:
    • 웹 페이지의 URL을 복사하여 웹 북마크 블록에 붙여넣으면 Notion이 자동으로 미리보기 정보를 가져와 표시합니다.

    스크린샷 가이드: 웹 북마크 URL 붙여넣기 및 결과

    • 목적: 웹 북마크 블록에 URL을 붙여넣었을 때 미리보기 정보가 표시되는 모습을 보여줍니다.
    • 가이드: 웹 북마크 블록에 URL을 붙여넣은 후, 웹 페이지의 제목, 설명, 미리보기 이미지가 표시된 화면을 캡처합니다.
    • 이미지 파일명: 8-7_web_bookmark_result.png

8.4. 코드 블록: 개발자를 위한 코드 스니펫 관리

프로그래밍 코드를 Notion 페이지에 깔끔하게 삽입하고 싶을 때 '코드 블록'을 사용합니다. 코드 하이라이팅 기능도 제공합니다.

  1. 코드 블록 추가:
    • 새로운 블록을 만들 위치에서 /를 입력하고 코드 또는 code를 입력하여 '코드' 블록을 선택합니다.

    스크린샷 가이드: 슬래시 명령어로 코드 블록 추가

    • 목적: 슬래시 명령어를 사용하여 코드 블록을 추가하는 과정을 보여줍니다.
    • 가이드: / 입력 후 '코드'를 검색하여 선택하는 모습을 캡처합니다.
    • 이미지 파일명: 8-8_add_code_block.png
  2. 코드 입력 및 언어 지정:
    • 코드 블록에 코드를 입력하거나 붙여넣습니다.
    • 코드 블록 우측 하단에서 프로그래밍 언어를 선택하면 해당 언어에 맞게 코드가 하이라이팅됩니다.

    스크린샷 가이드: 코드 입력 및 언어 지정

    • 목적: 코드 블록에 코드를 입력하고 프로그래밍 언어를 지정하는 모습을 보여줍니다.
    • 가이드: 코드 블록에 코드가 입력된 상태에서 언어 선택 드롭다운 메뉴를 클릭하여 언어를 선택하는 모습을 캡처합니다.
    • 이미지 파일명: 8-9_code_block_language.png
  3. 캡션 및 복사 버튼:
    • 코드 블록 아래에 캡션을 추가하여 코드에 대한 설명을 달 수 있습니다.
    • 코드 블록 우측 상단에는 '복사' 버튼이 있어 코드를 쉽게 복사할 수 있습니다.

    스크린샷 가이드: 코드 블록 캡션 및 복사 버튼

    • 목적: 코드 블록의 캡션과 복사 버튼을 보여줍니다.
    • 가이드: 캡션이 입력된 코드 블록과 우측 상단의 '복사' 버튼이 보이는 화면을 캡처합니다.
    • 이미지 파일명: 8-10_code_block_caption_copy.png

8.5. 파일 블록: 모든 종류의 파일을 Notion에 저장

문서, 스프레드시트, PDF, 압축 파일 등 어떤 종류의 파일이든 Notion 페이지에 직접 업로드하여 관리할 수 있습니다.

  1. 파일 블록 추가:
    • 새로운 블록을 만들 위치에서 /를 입력하고 파일 또는 file을 입력하여 '파일' 블록을 선택합니다.

    스크린샷 가이드: 슬래시 명령어로 파일 블록 추가

    • 목적: 슬래시 명령어를 사용하여 파일 블록을 추가하는 과정을 보여줍니다.
    • 가이드: / 입력 후 '파일'을 검색하여 선택하는 모습을 캡처합니다.
    • 이미지 파일명: 8-11_add_file_block.png
  2. 파일 업로드:
    • '파일 업로드' 버튼을 클릭하여 컴퓨터에 있는 파일을 선택하거나, 파일을 직접 드래그 앤 드롭하여 업로드합니다.

    스크린샷 가이드: 파일 업로드

    • 목적: 파일 블록에 파일을 업로드하는 과정을 보여줍니다.
    • 가이드: 파일 블록에 파일이 업로드된 후 파일명과 다운로드 버튼이 보이는 화면을 캡처합니다.
    • 이미지 파일명: 8-12_file_upload.png

강의 요약:

이번 8강에서는 Notion 페이지를 더욱 풍부하게 만들 수 있는 다양한 블록들을 살펴보았습니다. 비디오, 오디오, 웹 북마크, 코드 블록, 파일 블록을 활용하여 텍스트 외의 다양한 형태의 정보를 Notion에 통합하고 효과적으로 관리하는 방법을 배웠습니다. 이제 여러분은 Notion을 단순한 메모장을 넘어선 강력한 정보 관리 도구로 활용할 수 있습니다.
