5강: 필수 단축키와 마크다운 완전 정복

노션은 마우스 사용을 최소화하고 키보드만으로도 빠르고 효율적으로 작업할 수 있도록 다양한 단축키와 마크다운 문법을 지원합니다. 이 강의에서는 노션 사용의 생산성을 극대화할 수 있는 핵심 단축키와 마크다운 문법을 익히고, 이를 실제 작업에 적용하는 방법을 배웁니다.

학습 목표

  • 탐색과 편집에 가장 효과적인 10가지 핵심 키보드 단축키를 익히고 연습합니다.
  • 마크다운을 능숙하게 사용하여 마우스 없이 콘텐츠를 생성하고 서식을 지정합니다.

주요 개념

  • 블록 생성 단축키:
    • /: 모든 블록 유형 및 명령어를 검색하고 삽입
    • -, *, + + Space: 글머리 기호 목록
    • 1. + Space: 번호 매기기 목록
    • []: 할 일 목록
    • > + Space: 토글 목록
    • " + Space: 인용 블록
    • # + Space: H1 제목
    • ## + Space: H2 제목
    • ### + Space: H3 제목
    • ---: 구분선
  • 텍스트 서식 단축키:
    • Cmd/Ctrl + B: 굵게
    • Cmd/Ctrl + I: 기울임꼴
    • Cmd/Ctrl + U: 밑줄
    • Cmd/Ctrl + Shift + S: 취소선
    • Cmd/Ctrl + E: 코드
  • 블록 이동 및 복제:
    • Cmd/Ctrl + Shift + 방향키: 블록 이동
    • Cmd/Ctrl + D: 블록 복제
  • 마크다운 (Markdown): 텍스트 기반의 경량 마크업 언어로, 노션에서 텍스트 서식을 빠르게 적용하는 데 사용됩니다.

실습 과제

서식이 없는 일반 텍스트 문서를 제공하고, 수강생이 오직 키보드 단축키와 마크다운만을 사용하여 구조화되고 스타일이 적용된 노션 페이지로 변환하도록 합니다. 예를 들어, 다음 텍스트를 노션에 붙여넣고 단축키와 마크다운으로 서식을 적용해 보세요.
