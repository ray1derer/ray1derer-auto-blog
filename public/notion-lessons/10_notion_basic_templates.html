10강: 템플릿 활용의 시작 (Notion 기본 템플릿 불러오기 및 수정)

강의 목표: 이번 강의에서는 Notion이 제공하는 다양한 '템플릿'을 활용하여 Notion 사용을 더욱 쉽고 빠르게 시작하는 방법을 배웁니다. 기본 템플릿을 내 워크스페이스로 가져오고, 필요에 맞게 수정하는 방법을 익혀 보세요.

10.1. Notion 템플릿이란 무엇인가요? (미리 만들어진 페이지)

Notion '템플릿(Template)'은 특정 목적을 위해 미리 구성된 페이지 또는 데이터베이스입니다. 마치 요리할 때 레시피처럼, 템플릿은 Notion을 처음 사용하는 사람도 복잡한 설정 없이 바로 원하는 기능을 사용할 수 있도록 도와줍니다.

  • 시간 절약: 처음부터 페이지를 구성할 필요 없이, 이미 잘 만들어진 구조를 바로 사용할 수 있습니다.
  • 아이디어 얻기: 다른 사람들이 어떻게 Notion을 활용하는지 템플릿을 통해 아이디어를 얻을 수 있습니다.
  • 다양한 활용: 개인 생산성, 팀 협업, 교육, 디자인 등 다양한 분야의 템플릿이 제공됩니다.

10.2. Notion 기본 템플릿 갤러리 둘러보기

Notion은 공식적으로 다양한 기본 템플릿을 제공합니다.

  1. 템플릿 갤러리 접속:
    • 새 페이지를 만들 때 나타나는 옵션 중 '템플릿' 또는 'Templates'를 클릭합니다.

    스크린샷 가이드: 새 페이지 생성 시 템플릿 옵션

    • 목적: 새 페이지를 만들 때 템플릿을 선택할 수 있는 옵션을 보여줍니다.
    • 가이드: Notion에서 새 페이지를 생성한 후 나타나는 '템플릿' 또는 'Templates' 버튼을 캡처합니다.
    • 이미지 파일명: 10-1_new_page_template_option.png
  2. 템플릿 카테고리 탐색:
    • 템플릿 갤러리에서는 '개인', '팀', '교육', '디자인', '엔지니어링' 등 다양한 카테고리별로 템플릿을 탐색할 수 있습니다.

    스크린샷 가이드: 템플릿 갤러리 카테고리

    • 목적: 템플릿 갤러리의 다양한 카테고리를 보여줍니다.
    • 가이드: 템플릿 갤러리 화면에서 왼쪽 사이드바의 카테고리 목록을 캡처합니다.
    • 이미지 파일명: 10-2_template_gallery_categories.png
  3. 템플릿 미리보기:
    • 원하는 템플릿을 클릭하면 해당 템플릿의 미리보기를 확인할 수 있습니다.

    스크린샷 가이드: 템플릿 미리보기

    • 목적: 특정 템플릿을 클릭했을 때 나타나는 미리보기 화면을 보여줍니다.
    • 가이드: 템플릿 미리보기 화면에서 템플릿의 구성과 내용을 캡처합니다.
    • 이미지 파일명: 10-3_template_preview.png

10.3. 원하는 템플릿을 내 워크스페이스로 가져오기

마음에 드는 템플릿을 찾았다면, 간단하게 내 워크스페이스로 가져올 수 있습니다.

  1. 템플릿 가져오기:
    • 템플릿 미리보기 화면에서 우측 상단의 '템플릿 가져오기' 또는 'Get template' 버튼을 클릭합니다.

    스크린샷 가이드: 템플릿 가져오기 버튼

    • 목적: 템플릿 미리보기 화면에서 '템플릿 가져오기' 버튼을 보여줍니다.
    • 가이드: 템플릿 미리보기 화면의 우측 상단에 있는 '템플릿 가져오기' 버튼을 캡처합니다.
    • 이미지 파일명: 10-4_get_template_button.png
  2. 워크스페이스에 추가된 템플릿 확인:
    • 템플릿이 여러분의 Notion 워크스페이스에 새로운 페이지로 추가됩니다. 사이드바에서도 확인할 수 있습니다.

    스크린샷 가이드: 워크스페이스에 추가된 템플릿

    • 목적: 가져온 템플릿이 Notion 워크스페이스에 새로운 페이지로 추가된 모습을 보여줍니다.
    • 가이드: 사이드바에 새로 추가된 템플릿 페이지와 해당 페이지가 열린 화면을 캡처합니다.
    • 이미지 파일명: 10-5_template_added_to_workspace.png

10.4. 가져온 템플릿을 내 필요에 맞게 수정하기

가져온 템플릿은 여러분의 필요에 따라 자유롭게 수정할 수 있습니다.

  1. 블록 추가/삭제/수정:
    • 템플릿 내의 텍스트, 이미지, 체크리스트 등 모든 블록을 일반 페이지처럼 수정할 수 있습니다. 필요 없는 블록은 삭제하고, 새로운 블록을 추가하여 내용을 보완합니다.

    스크린샷 가이드: 템플릿 블록 수정

    • 목적: 가져온 템플릿 내의 블록을 수정하는 모습을 보여줍니다.
    • 가이드: 템플릿의 특정 텍스트를 수정하거나, 기존 블록을 삭제하고 새로운 블록을 추가하는 모습을 캡처합니다.
    • 이미지 파일명: 10-6_edit_template_blocks.png
  2. 속성 변경 (데이터베이스 템플릿의 경우):
    • 만약 가져온 템플릿이 데이터베이스를 포함하고 있다면, 데이터베이스의 속성(Property)을 변경하거나 새로운 속성을 추가할 수 있습니다. (예: '프로젝트 관리' 템플릿에서 '담당자' 속성을 추가)

    스크린샷 가이드: 데이터베이스 템플릿 속성 변경

    • 목적: 데이터베이스 템플릿의 속성을 변경하는 모습을 보여줍니다.
    • 가이드: 템플릿 내 데이터베이스의 속성 설정 메뉴를 열어 속성 유형을 변경하거나 새로운 속성을 추가하는 모습을 캡처합니다.
    • 이미지 파일명: 10-7_edit_template_db_properties.png
  3. 템플릿 이름 변경:
    • 가져온 템플릿 페이지의 제목을 여러분이 원하는 이름으로 변경할 수 있습니다.

    스크린샷 가이드: 템플릿 페이지 이름 변경

    • 목적: 가져온 템플릿 페이지의 제목을 변경하는 모습을 보여줍니다.
    • 가이드: 템플릿 페이지의 제목을 클릭하여 새로운 이름으로 수정하는 모습을 캡처합니다.
    • 이미지 파일명: 10-8_rename_template_page.png

강의 요약:

이번 10강에서는 Notion이 제공하는 다양한 '템플릿'을 활용하여 Notion 사용을 쉽고 빠르게 시작하는 방법을 배웠습니다. 템플릿 갤러리를 탐색하고, 원하는 템플릿을 내 워크스페이스로 가져온 후, 필요에 맞게 수정하는 방법을 익혔습니다. 템플릿은 Notion을 효율적으로 활용하는 강력한 도구입니다.
