11강: 고급 블록 활용 - 버튼, 동기화 블록, 목차

노션은 단순한 텍스트 블록을 넘어, 워크플로우를 자동화하고 콘텐츠를 효율적으로 관리할 수 있는 고급 블록들을 제공합니다. 이 강의에서는 '버튼', '동기화 블록', '목차'와 같은 유용한 블록들을 활용하여 노션 사용 경험을 한 단계 업그레이드하는 방법을 배웁니다.

학습 목표

  • 버튼 블록을 사용하여 간단하고 반복적인 작업을 자동화합니다.
  • 동기화 블록을 사용하여 여러 페이지에 걸쳐 콘텐츠를 일관성 있게 유지합니다.
  • 긴 문서에 대해 자동으로 탐색 가능한 목차를 생성합니다.

주요 개념

  • 버튼 블록 (Button Block): /버튼을 입력하여 생성합니다. 클릭 한 번으로 페이지 추가, 페이지 편집, 알림 보내기 등 미리 정의된 작업을 수행할 수 있습니다. 반복적인 작업을 자동화하는 데 매우 유용합니다.
  • 동기화 블록 (Synced Block): /동기화 블록을 입력하여 생성합니다. 이 블록 안에 있는 콘텐츠는 복사하여 다른 여러 페이지에 붙여넣을 수 있으며, 원본 블록을 수정하면 모든 복사본이 자동으로 업데이트됩니다. 공통 메뉴, 푸터, 중요 공지사항 등을 여러 페이지에 일관성 있게 유지할 때 사용합니다.
  • 목차 블록 (Table of Contents Block): /목차를 입력하여 생성합니다. 페이지 내의 모든 제목(H1, H2, H3 등)을 자동으로 인식하여 클릭 가능한 목차를 만들어 줍니다. 긴 문서에서 특정 섹션으로 빠르게 이동하는 데 유용합니다.
  • 브레드크럼 블록 (Breadcrumb Block): /브레드크럼을 입력하여 생성합니다. 현재 페이지의 계층 구조를 보여주어 사용자가 워크스페이스 내에서 자신의 위치를 파악하고 상위 페이지로 쉽게 이동할 수 있도록 돕습니다.

실습 과제

  • 버튼: '새 일일 저널 항목' 템플릿을 페이지에 삽입하는 버튼을 만듭니다. (예: 버튼 클릭 시 미리 정의된 '일일 저널' 템플릿 페이지가 생성되도록 설정)
  • 동기화 블록: 주요 페이지(홈, 프로젝트, 목표)로 연결되는 링크가 포함된 '탐색 메뉴' 블록을 생성합니다. 이 블록을 복사하여 최소 두 개 이상의 다른 페이지에 동기화합니다.
