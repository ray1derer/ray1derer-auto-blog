Notion 완전 정복: 일상 속 문제 해결부터 시작하는 스마트 워크스페이스 구축

5강: Notion과 외부 서비스 연동의 맛보기 (Google Calendar, Email, Web Clipper)

강의 목표: 이번 강의에서는 Notion을 Google Calendar, 이메일 등 외부 서비스와 연동하여 Notion의 활용 범위를 넓히는 방법을 배웁니다. Notion이 어떻게 여러분의 디지털 허브가 될 수 있는지 경험해 보세요.

5.1. Notion 위젯과 임베드: 외부 콘텐츠를 내 페이지에!

Notion의 '임베드(Embed)' 기능은 외부 웹사이트나 서비스를 Notion 페이지 안에 직접 삽입하여 보여주는 강력한 기능입니다. 이를 통해 Notion을 떠나지 않고도 다양한 정보를 한눈에 확인하고 관리할 수 있습니다.

  1. 임베드 블록 추가:
    • 새로운 블록을 만들 위치에서 /를 입력하고 임베드 또는 embed를 입력하여 '임베드' 블록을 선택합니다.

    스크린샷 가이드: 슬래시 명령어로 임베드 블록 추가

    • 목적: 슬래시 명령어를 사용하여 임베드 블록을 추가하는 과정을 보여줍니다.
    • 가이드: / 입력 후 '임베드'를 검색하여 선택하는 모습을 캡처합니다.
    • 이미지 파일명: 5-1_add_embed_block.png
  2. Google Calendar 임베드하기:
    • Google Calendar를 Notion 페이지에 삽입하여 Notion에서 바로 일정을 확인하고 싶을 때 유용합니다.
    • Google Calendar에서 공유 가능한 링크 얻기:
      • Google Calendar에 접속하여 설정(톱니바퀴 아이콘)으로 이동합니다.
      • '내 캘린더 설정'에서 공유하고 싶은 캘린더를 선택합니다.
      • '캘린더 설정'에서 '공유 가능한 링크 가져오기' 또는 '공개 URL'을 찾아 복사합니다. (이때 '공개 캘린더로 만들기' 옵션을 활성화해야 할 수 있습니다.)

      스크린샷 가이드: Google Calendar 공유 가능한 링크 복사

      • 목적: Google Calendar 설정에서 공유 가능한 링크를 복사하는 과정을 보여줍니다.
      • 가이드: Google Calendar 설정 화면에서 특정 캘린더의 '공유 가능한 링크' 또는 '공개 URL'이 보이는 부분을 캡처합니다.
      • 이미지 파일명: 5-2_google_calendar_share_link.png
    • Notion에 링크 붙여넣기:
      • Notion 페이지의 임베드 블록에 복사한 Google Calendar 링크를 붙여넣고 '임베드 링크' 버튼을 클릭합니다.

      스크린샷 가이드: Notion에 Google Calendar 임베드

      • 목적: Notion 페이지에 Google Calendar가 성공적으로 임베드된 모습을 보여줍니다.
      • 가이드: Notion 페이지 안에 Google Calendar가 캘린더 형태로 표시된 화면을 캡처합니다.
      • 이미지 파일명: 5-3_notion_embed_google_calendar.png

5.2. Notion Web Clipper: 웹 페이지를 Notion으로 스크랩!

Notion Web Clipper는 웹 브라우저 확장 프로그램으로, 보고 있는 웹 페이지를 Notion 페이지로 쉽게 저장할 수 있게 해줍니다. 중요한 기사, 참고 자료, 레시피 등을 Notion에 바로 스크랩하여 나중에 편리하게 찾아볼 수 있습니다.

  1. Notion Web Clipper 설치:
    • 사용하는 웹 브라우저(Chrome, Firefox, Safari 등)의 확장 프로그램 스토어에서 'Notion Web Clipper'를 검색하여 설치합니다.

    스크린샷 가이드: Notion Web Clipper 설치 페이지

    • 목적: 웹 스토어에서 Notion Web Clipper를 검색하여 설치하는 페이지를 보여줍니다.
    • 가이드: Chrome 웹 스토어 등에서 'Notion Web Clipper' 검색 결과 페이지를 캡처합니다.
    • 이미지 파일명: 5-4_web_clipper_install_page.png
  2. 웹 페이지 스크랩하기:
    • 스크랩하고 싶은 웹 페이지를 열고, 브라우저의 확장 프로그램 아이콘에서 Notion Web Clipper를 클릭합니다.
    • 저장할 Notion 워크스페이스와 페이지를 선택하고 '페이지에 저장' 버튼을 클릭합니다.

    스크린샷 가이드: 웹 페이지 스크랩 과정

    • 목적: Notion Web Clipper를 사용하여 웹 페이지를 Notion으로 스크랩하는 과정을 보여줍니다.
    • 가이드: 웹 페이지가 열려 있는 상태에서 Notion Web Clipper 아이콘을 클릭했을 때 나타나는 팝업 창과 저장 옵션을 캡처합니다.
    • 이미지 파일명: 5-5_web_clipper_usage.png
  3. Notion에서 스크랩된 페이지 확인:
    • Notion 워크스페이스에서 스크랩한 페이지를 확인합니다. 웹 페이지의 내용이 Notion 페이지로 깔끔하게 정리되어 들어와 있을 것입니다.

    스크린샷 가이드: Notion에 스크랩된 웹 페이지

    • 목적: Notion에 스크랩된 웹 페이지의 내용을 보여줍니다.
    • 가이드: Notion에 저장된 웹 페이지가 열려 있는 화면을 캡처합니다.
    • 이미지 파일명: 5-6_web_clipper_result.png

5.3. 이메일 내용을 Notion으로 저장하기 (간단한 방법)

Notion은 이메일 클라이언트가 아니지만, 중요한 이메일 내용을 Notion으로 가져와 관리할 수 있는 몇 가지 방법이 있습니다.

  1. 이메일 내용 복사-붙여넣기:
    • 가장 간단한 방법입니다. 이메일 내용을 복사하여 Notion 페이지에 붙여넣으면, Notion이 자동으로 텍스트 블록으로 변환해 줍니다.

    스크린샷 가이드: 이메일 내용 복사-붙여넣기

    • 목적: 이메일 내용을 복사하여 Notion 페이지에 붙여넣는 과정을 보여줍니다.
    • 가이드: 이메일 클라이언트에서 이메일 내용을 드래그하여 복사하는 모습과, Notion 페이지에 붙여넣은 후의 화면을 캡처합니다.
    • 이미지 파일명: 5-7_email_copy_paste.png
  2. 이메일 전달 기능 활용 (고급):
    • 일부 이메일 서비스는 특정 이메일 주소로 메일을 전달하면 Notion 페이지로 자동 저장되는 기능을 제공하기도 합니다. (Notion의 '데이터베이스 이메일 주소' 기능과 연동) 이 기능은 캘린더나 할 일 데이터베이스에 이메일을 자동으로 추가할 때 유용합니다.

    스크린샷 가이드: Notion 데이터베이스 이메일 주소 확인

    • 목적: Notion 데이터베이스의 '이메일 주소'를 확인하는 방법을 보여줍니다.
    • 가이드: Notion 데이터베이스의 '...' 메뉴에서 '이메일 주소 가져오기' 또는 유사한 옵션을 클릭했을 때 나타나는 이메일 주소를 캡처합니다.
    • 이미지 파일명: 5-8_notion_db_email_address.png

강의 요약:

이번 5강에서는 Notion을 Google Calendar, 웹 페이지, 이메일 등 외부 서비스와 연동하는 방법을 배웠습니다. '임베드' 기능을 통해 Notion 페이지 안에서 다양한 정보를 확인하고, '웹 클리퍼'를 통해 웹 페이지를 Notion으로 쉽게 스크랩할 수 있음을 확인했습니다. Notion은 이렇게 외부 서비스와 연결되어 여러분의 모든 정보를 통합 관리하는 강력한 허브가 될 수 있습니다.

6강: Notion의 '블록' 개념 이해하기 (왜 모든 것이 블록일까요?)

  • 블록의 종류와 역할 (텍스트, 미디어, 데이터베이스, 임베드 등)

  • 슬래시 명령어(/)를 통한 블록 추가의 편리함과 다양한 블록 탐색
